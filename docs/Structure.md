# Structure

#### dlgo.py

使用 dlgo 的入口。

#### train.py

實做整訓練管線，也是訓練的入口。

#### board.py

實做整個棋盤的演算法，包括規則和神經網路的輸入。

#### config.py

棋盤大小和網路結構相關設定。

#### gtp.py

實做 GTP 協議的界面。

#### gui.py

實做圖形界面。

#### mcts.py

實做蒙地卡羅樹搜索。

#### network.py

實做神經網路結構。

#### transformer.py

實做 transformer 網路，它是神經網路實做的一部分。

#### sgf.py

SGF 格式檔案的解析器。

#### time_control.py

樹搜索的時間控制器，符合標準 GTP 協議的格式。
